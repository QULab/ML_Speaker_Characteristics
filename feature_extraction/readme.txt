- This folder contains necessary code for feaure extraction with OpenSMILE.
- The last version needs to be downloaded from https://audeering.com/technology/opensmile/ 
- I used OpenSMILE version 2.3.0 on Windows 10
- Need to adjust the paths to wav files and to the configuration file in the python code.
- Need to copy the following files to this folder:
	SMILExtract_Release.exe
	openSmileLib_Release.dll
- Features are writen to the csvoutput file in the /extracted_features folder
- If a different configuration file is desired, change confFile in code
